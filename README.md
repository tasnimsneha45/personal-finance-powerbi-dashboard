# Personal Finance Dashboard (Excel + Power BI)

## 📊 Project Overview
This project analyzes personal financial data to track income, expenses, and savings trends using Excel and Power BI.

## 🛠 Tools Used
- Microsoft Excel (Data cleaning, VLOOKUP/XLOOKUP, Pivot Tables)
- Power BI (Data modeling, DAX measures, interactive dashboard)

## 📈 Features
- Income vs Expense tracking
- Category-wise spending analysis
- Monthly spending trends
- Net savings calculation
- Interactive filtering by category

## 📌 Key Skills Demonstrated
- Data analysis
- Data visualization
- Data modeling
- Business KPI creation
- Dashboard design
